Claude Code 配置指南

目前 claude code 价格：¥300/月 有$100–$150/天

✨ 适用于 Windows（含路径示例），其他平台亦可参考

  1. 📦 安装命令行工具

    使用 npm 全局安装 Claude Code：

    npm install -g @anthropic-ai/claude-code

  2. ⚙️ 创建或编辑配置文件

    在用户目录下新建（或编辑）设置文件：

    %USERPROFILE%\.claude\settings.json
    {
  "env": {
    "ANTHROPIC_AUTH_TOKEN": "you-key",
    "ANTHROPIC_BASE_URL": "https://mfai.xiaoyulove.xyz",
    "CLAUDE_CODE_DISABLE_NONESSENTIAL_TRAFFIC": 1
  },
  "permissions": {
    "allow": [],
    "deny": []
  },
  "apiKeyHelper": "echo 'you-key'"
}
    ⚠️
    请妥善保管密钥，避免公开分享；如有需要，建议替换为你自己的有效密钥与可信的 API 地址。

  3. ✅ 验证安装

    保存配置后，执行下列命令查看是否可用：

    claude --version

    若输出版本号，即表示安装成功。接下来可以在任意目录运行：

    claude
✨ 完成以上步骤后即可在命令行中使用 claude-code
💡 小提示：如遇网络/证书/权限问题，优先检查代理与环境变量

Codex 配置指南

Codex 是一款强大的 AI 代码辅助工具，支持多种编程语言和框架

🔥 简单配置，即刻使用

  1. 📦 安装 OpenAI Codex CLI 工具

    使用 npm 或 brew 安装 Codex 命令行工具：

    # 使用 npm 全局安装（推荐）
npm install -g @openai/codex

# 或使用 Homebrew (macOS/Linux)
brew install codex
    💡
    确保已安装 Node.js (npm 方式) 或 Homebrew (brew 方式)。安装完成后可在任意目录使用 codex 命令。

  2. ⚙️ 配置 config.toml 文件

    在 ~/.codex/config.toml 创建配置文件：

    model_provider = "xiaoyumod"
model = "gpt-5"  # 可更改为 model = "gpt-5-codex"
model_reasoning_effort = "high"
disable_response_storage = true

[model_providers.xiaoyumod]
name = "xiaoyumod"
base_url = "https://mfai.xiaoyulove.xyz/v1"
wire_api = "responses"
requires_openai_auth = true
    ℹ️
    此配置使用 xiaoyumod 作为模型提供商，支持 gpt-5 和 gpt-5-codex 模型。推理级别设为 high 可获得更好的代码质量。

  3. 🔑 配置 auth.json 文件

    在 ~/.codex/auth.json 创建认证文件：

    {
  "OPENAI_API_KEY": "your-api-key-here"
}
    ⚠️
    请将 your-api-key-here 替换为你的真实 API 密钥。密钥需要从相应的服务提供商处获取，请妥善保管。

  4. ✅ 验证安装

    运行以下命令验证 Codex 是否正确安装：

    codex --version

  • 🚀 开始使用

    在终端中启动 Codex 交互模式：

    codex

    或者直接提问：

    codex "如何在 Python 中读取 JSON 文件？"
    ⭐
    Codex 支持多轮对话、代码生成、调试辅助等功能。你可以在项目目录中使用它来获取上下文相关的帮助。
    • 🎉 配置完成后，即可享受 AI 辅助编程的强大功能！
